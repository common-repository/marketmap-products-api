== Description ==
&#x202b; این پلاگین برای دریافت لیست محصولات ووکامرس و به روز رسانی سریع قیمت در [مارکت مپ](https://marketmap.org) طراحی شده است.
[marketmap.org](https://marketmap.org)
## نیازمندی ها
* وردپرس نسخه ۵.۲.۰ به بالا
* ووکامرس نسخه ۴.۰.۰ به بالا
## نحوه استفاده
* &#x202b;افزونه را نصب نمایید.
* &#x202b;افزونه را فعال نمایید.
* &#x202b;به پشتیبانی مارکت مپ اطلاع دهید.

## خروجی
&#x202b; اطلاعات کلی محصولات و دسته بندی ها در خروجی وب سرویس نمایش داده می شود:
* &#x202b; id
* &#x202b; title
* &#x202b; price
* &#x202b; content
* &#x202b; stock_status
* &#x202b; link
* &#x202b; product_category
* &#x202b; variations
* &#x202b; tags
* &#x202b; attribute

== Installation ==

1. &#x202b;افزونه را نصب نمایید.
2. &#x202b;افزونه را از قسمت افزونه ها فعال نمایید.
